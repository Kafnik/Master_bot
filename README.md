## 🎄 О боте

**Новогодний Игровой Бот** - это праздничный Telegram-бот с коллекцией увлекательных игр для создания новогоднего настроения! Бот предлагает различные развлечения, игры и пасхалки в духе зимних праздников.

## ✨ Особенности

### 🎮 **Игры**
- 🎯 **Угадай число** - классическая игра с ограниченным количеством попыток
- 🎡 **Новогоднее колесо** - крутите колесо и получайте праздничные события
- ❄️ **Снежинки** - ловите снежинки и соревнуйтесь в количестве
- 🎁 **Новогодняя пасхалка** - секретная команда `/new_year` с подарками!

### 🛠 **Функции**
- ✅ Полная настройка прав администратора
- 🔧 Включение/выключение бота в любое время
- 🎨 Праздничный интерфейс с новогодней тематикой
- 📊 Система попыток и подсчета очков
- 🔄 Интерактивное меню с кнопками

## 🚀 Установка

### Быстрый старт
```bash
# Клонировать репозиторий
git clone https://github.com/ваш-репозиторий/new-year-bot.git

# Перейти в директорию
cd new-year-bot

# Установить зависимости
pip install -r requirements.txt

# Настроить конфигурацию
cp config.py.example config.py
# Отредактировать config.py, добавив токен бота

# Запустить бота
python bot.py
